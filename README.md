## 🔧 Реализованные функции

### 1. `double* full_elements(double *ptr_array, int n)`
- **Назначение**: Заполнение массива значениями
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: указатель на заполненный массив
