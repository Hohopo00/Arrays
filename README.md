## Реализованные функции

### 1. double* full_elements(double* ptr_array, int n, double x)
- Назначение: Заполнение массива значениями, вычисленными функцией calc()
- Параметры:
  - ptr_array - указатель на массив
  - n - количество элементов
  - x - начальное значение для вычислений
- Возвращает: указатель на заполненный массив

### 2. int ptr_elements(double* ptr_array, int n)
- Назначение: Вывод элементов массива на консоль
- Параметры:
  - ptr_array - указатель на массив
  - n - количество элементов
- Возвращает: количество выведенных элементов

### 3. double* calc_elements(double* ptr_array, int n)
- Назначение: Умножение каждого элемента массива на 2
- Параметры:
  - ptr_array - указатель на массив
  - n - количество элементов
- Возвращает: указатель на измененный массив

### 4. double calc(double x)
- Назначение: Вычисление значения функции f(x) = x³ - 4x² + 2
- Параметры:
  - x - значение аргумента
- Возвращает: результат вычисления

### 5. double sum_elements(double* ptr_array, int begin, int end)
- Назначение: Сумма элементов массива в заданном диапазоне
- Параметры:
  - ptr_array - указатель на массив
  - begin - начальный индекс
  - end - конечный индекс
- Возвращает: сумму элементов

### 6. int find_element(double* ptr_array, int n, double element)
- Назначение: Поиск элемента в массиве с заданной точностью
- Параметры:
  - ptr_array - указатель на массив
  - n - количество элементов
  - element - искомое значение
- Возвращает: 1 если найден, -1 если не найден
